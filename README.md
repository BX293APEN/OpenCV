# How to OpenCV
## cv::Mat
| Method | Description |
| ------ | ----------- |
| `.rows` | 行数 |
| `.cols` | 列数 |
| `.dims` | 次元数 |
| `.channels()` | チャンネル数 |
| `.elemSize()` | 1要素のサイズ [Byte] |
| `.elemSize1()` | 1要素内の1ch分のサイズ [Byte] |
| `.total()` | 要素の総数 |
| `.step` | ステップ数 [Byte] |
| `.data[]` | データ配列 |
